# Others Game Assets Repository

This repository contains organized visual assets for web development and game promotion. All assets are categorized by their roles to ensure easy access for developers.

## 📁 Folder Structure

| Folder | Description |
| :--- | :--- |
| `backgrounds/` | UI backgrounds for various game types (Live Arena, Lottery, Sports, etc.) |
| `game-assets/` | Specific game-related assets including `jackpots` and `tournaments`. |
| `icons/` | General UI icons and browser-specific icons. |
| `logos/platforms/` | Official logos for various gaming platforms (JDB, JILI, PP, etc.) |
| `payments/` | Payment-related assets (e.g., USDT). |
| `promotions/` | Promotion banners and event-specific assets (Daily Check-in, Free Spins, etc.) |

## 🚀 Usage for Developers

Developers can link to these assets directly in their projects. Most images are in **WebP** format for optimized web performance, with some **PNG** and **SVG** files where transparency or scalability is required.

### Example:
```html
<img src="https://raw.githubusercontent.com/ccorryxx-bot/others-game-assets/main/logos/platforms/JILI-logo.webp" alt="JILI Logo">
```

## 🛠 Management
The repository is maintained by organizing files into roles. When adding new assets, please ensure they are placed in the appropriate category folder.
